001 : Reyta the Twitter bot

Released on 20-01-2021
Reyta is a bot developed using the Twitter API and Tweepy API it does a variety of functions such as Retweet and like all tweets in the feed without exceeding the limit set by twitter as well as following back every user that follows Reyta.
Reyta is set to be further developed to extend its features to Instagram
